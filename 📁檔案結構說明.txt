│   └─ 常見問題
│
├── 📄 README.md                   # 📝 專案說明（Markdown 格式）
│   └─ 適合放在 GitHub 或技術文件
│
├── 📄 📦分享說明.txt              # 🎁 給分享者的指南
│   ├─ 如何打包分享

│   └─ 注意事項

使用者需要自行建立的檔案：
========================================

📄 .env                             # ⚙️ 環境變數檔（不要分享）
   └─ 包含個人的 YouTube API Key


執行後自動產生的檔案/資料夾：
========================================

📁 .venv/                          # 🐍 Python 虛擬環境
   └─ 由「安裝.bat」自動建立


分享前要刪除的檔案（如果存在）：
========================================

❌ .env                            # 包含你的 API Key
❌ .venv/                          # 虛擬環境（每個人自己建立）
❌ __pycache__/                    # Python 快取
❌ *.pyc                           # 編譯檔


========================================
檔案用途說明
========================================

🔧 安裝.bat
  • 第一次使用時執行
  • 檢查 Python 是否已安裝
  • 建立虛擬環境
  • 安裝所有必要套件

🚀 啟動.bat
  • 每次使用時執行
  • 啟動虛擬環境
  • 執行 Flask 網頁伺服器
  • 自動開啟瀏覽器（或提示網址）

📖 使用說明.txt
  • 給一般使用者的完整教學
  • 包含圖文並茂的步驟說明
  • 常見問題解答
  • 適合不熟悉程式的人閱讀

📝 README.md
  • 給開發者或技術人員的說明
  • Markdown 格式
  • 適合放在 GitHub
  • 包含技術細節

🎁 📦分享說明.txt
  • 給要分享程式的人看
  • 如何正確打包
  • 分享檢查清單
  • 進階打包選項

📦 requirements.txt
  • Python 套件依賴清單
  • 由 pip 自動讀取安裝
  • 確保所有人使用相同版本的套件

⚙️ .env.example
  • 環境變數範本
  • 使用者需複製為 .env
  • 包含詳細的註解說明
  • 不包含真實的 API Key

💻 app.py
  • Flask 主程式
  • 處理所有後端邏輯
  • YouTube API 整合
  • 約 900+ 行程式碼

🌐 index.html
  • 前端網頁介面
  • 使用 Bootstrap 5 設計
  • 響應式設計（手機/平板/電腦）
  • 包含 JavaScript 互動功能

========================================
完整使用流程
========================================

【分享者】
1. 壓縮 distribution 資料夾
2. 確認已刪除 .env 和 .venv
3. 分享壓縮檔給使用者

【使用者】
1. 解壓縮資料夾
2. 雙擊「安裝.bat」
3. 取得 YouTube API Key（參考使用說明.txt）
4. 複製 .env.example 為 .env
5. 編輯 .env 填入 API Key
6. 雙擊「啟動.bat」
7. 開啟瀏覽器 http://127.0.0.1:5000
8. 開始搜尋！
